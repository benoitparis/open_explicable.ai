Explainable machine learning

Wed, Nov 1, 2017

While achieving great predictive power, machine learning models can be really opaque for their users; Be it a data science team, a marketing executive, a salesperson, or even the end user.

DARPA has recently launched a program aimed just at that: Explainable Artificial Intelligence (XAI). While DARPA’s approach is in specifically targeting black-box models, there is plenty of insights in inspecting the internals of the model at hand.

The aim of this project is to build support for explorations of model parameters, to provide humans an understanding of decisions taken.
